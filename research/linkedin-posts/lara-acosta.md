# Lara Acosta - LinkedIn Posts

## Post 1 - [Posted a month ago]
I grew to 340k followers on LinkedIn by posting 4x/week.

You don't need to post daily here to grow.
The new LinkedIn algorithm updates also show that.

Your content gets rewarded via quality, not quantity. 

Here's a simple structure you can use this week:

I call it the '4-3-2-1' Framework: 

4 posts a week (non-negotiable)
3 content pillars (educate, storytell and sell)
2 types of audiences (followers and customers)
1 lead magnet every week (to convert to email).

That's it. That's your system for the week. 

Pick a pillar. Solve a problem. Use an 8 word hook.

## Post 2 - [Posted a month ago]
I keep on saying this. Repeat yourself, please. 

This is how you grow a following. 
This is how you become a thought leader.
This is how you stand out in a saturated niche.

Own a niche by talking about it every single time. 

I use a content routine every week that never fails:

1. Pick the topic you want to be known for.
2. Find 3-5 subtopics that tie into the main topic.
3. Pick 1 to talk about for 4-days of the week.
4. Use different angles (education or storytelling).
5. Repeat the following week. 

This is simple, but it's what's built me a $2M+ business.

All because I post about the same thing every week. 

Repetition builds reputation over time.

## Post 3 - [Posted two months ago]
I've written 1,023 viral LinkedIn posts using this...

LinkedIn loves this framework; it works in any niche. 

1. Start with a story to grab their attention. 
2. Follow up with a key lesson to retain it.
3. Expand with actionable advice so they save it. 
4. End with a question so they engage with it faster.

That's it. Simple, but it works. 

Length doesn't matter, value per line does. 

This post is a short example of how to do it.
